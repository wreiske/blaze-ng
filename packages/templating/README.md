# @blaze-ng/templating

Blaze-NG templating meta-package. Bundles the templating runtime and compiler into a single dependency.

## Installation

```bash
npm install @blaze-ng/templating
```

## Usage

```ts
import { Template, __define__ } from '@blaze-ng/templating';
```

## What's Included

Re-exports from:

- `@blaze-ng/templating-runtime` — Template registration, helpers, events, lifecycle
- `@blaze-ng/templating-compiler` — HTML-to-JS compilation
